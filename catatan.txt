clean
===================
./gradlew clean



rebuild
====================
./gradlew clean build



run
===================
./gradlew run
./gradlew build run
./gradlew clean build run



debug
==================
./gradlew :launcher:run --debug-jvm
kemudian jalankan debugger vscode

atau bisa juga menggunakan perintah
./gradlew run -Pdebug



build exe (image)
=====================
./gradlew :launcher:jpackageImage


build installer
========================
./gradlew :launcher:jpackage
